# bad_bytes

Vite + React + TypeScript website.

Live:

- https://bcrnic.github.io/bad_bytes/

## Tech

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Getting started

Requirements:

- Node.js (recommended: Node 20)
- npm

Install dependencies:

```sh
npm install
```

Run dev server:

```sh
npm run dev
```

Build for production:

```sh
npm run build
```

Preview production build locally:

```sh
npm run preview
```
